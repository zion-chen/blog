# 🔰 关于我
我是陈自涵，西安电子科技大学2020级软件工程专业在读。
爱好是 ***地理，行政区划，交通规划，汉语方言。***
**梦想** 是死之前走遍中国每一个县级行政区。
我想去看漠河的极光，想去看喀什到底是不是人均高加索面孔，想去看昆山常熟宜兴吴江到底有多繁荣，想去看防城港桥下是不是天天都有拉猪脚的大飞。
我希望我可以成为一个 **博学** 的人，我坚信，只有经历过才有谈论的资本。

# 🌈 我的推荐
以下漫画、影视作品、游戏均是自初中以来，在我极其严苛的筛选下，我认为可以吸引到我的。
对我影响最深的就是 ***《少女与战车》*** 了。这部番我2013开始看TV版，中学时期慢慢追到了最终章，没想到大学都要毕业了还没等到最终章第四集，可能等最终章完结我都可以和我闺女一起看了。
此外，***《古见同学有交流障碍症》*** 这部番让我有了“好想谈恋爱！”这样的想法。2023年四月末的某天，我一晚上刷完了两季动画，在去武汉的高铁上四个小时看到了漫画连载最新章节，虽然他们说是工业糖精，但谁不喜欢甜甜的恋爱番呢。
